The complete RESTful API for EDDB

Donate

Currently EDDB is the greatest treasure of information of the galaxy and it is a user friendly tool for many pilots. But it lacks an API for developers. Elite BGS is an attempt to fill this gap by providing an API for EDDB data sources.

API

Access
Consumers must make a GET request at any endpoint using Basic HTTP authentication with username guest and password secret.
Endpoints

Bodies

http://elitebgs.kodeblox.com/api/eddb/v1/bodies?<params>

Commodities

http://elitebgs.kodeblox.com/api/eddb/v1/commodities/id/<commodity-id>

Factions
